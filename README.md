# ReviewIQ

AI-powered restaurant review intelligence.

## Overview

ReviewIQ is a personal project designed to integrate concepts from:

- Natural Language Processing (NLP)
- Large Language Models (LLMs)
- Data Analysis
- Business Intelligence
- Prompt Engineering

The project analyzes restaurant reviews and transforms unstructured customer feedback into actionable business insights.

## Goals

- Apply NLP concepts in a real-world scenario.
- Learn Prompt Engineering.
- Work with local LLMs using LM Studio.
- Build end-to-end data analysis pipelines.
- Create professional dashboards using Power BI.
- Document the learning journey.
